# jQueryUI - Accordion Addon

This addon provides actions to automate jQuery Accordion control \(Displays collapsible content panels for presenting information in a limited amount of space\). For more information about the jQuery Accordion, visit: [https://jqueryui.com/accordion/](https://jqueryui.com/accordion/)

### Available Actions

`Click section by index` - Clicks section found by it's index

`Click section by title` - Clicks section found by it's title
